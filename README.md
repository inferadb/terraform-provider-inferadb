# Terraform Provider for InferaDB

Infrastructure as code provider for deploying and managing InferaDB resources.

## Status

**In Development** - This provider is not yet available for production use.

## Planned Features

- InferaDB server deployment
- Vault and policy configuration
- Relationship management
- Multi-tenancy setup
